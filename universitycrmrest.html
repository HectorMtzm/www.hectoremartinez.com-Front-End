UniversityCRM REST API

Here you can find all the endpoints and acceptable HTTP request methods they support

GET

To retrive the students list:

http://www.hectoremartinez.com/portfolio/universitycrm/api/students

To retrieve a specific student, use the student ID:

http://www.hectoremartinez.com/portfolio/universitycrm/api/students/{id}

To retrieve the classes of a specific student:

http://www.hectoremartinez.com/portfolio/universitycrm/api/students/{id}/courses
POST

To create a new student, use the following endpoint and JSON format in the body:

http://www.hectoremartinez.com/portfolio/universitycrm/api/students

        "city": "Richardson",
        "state": "Texas",

}

To register a student to a class, use the following endpoint and JSON format in the body:

http://www.hectoremartinez.com/portfolio/universitycrm/api/students/{id}/courses 
{
    "id": {Course ID}
}
